# Setup

1. Download http://www.apache.org/dist/mahout/0.7/mahout-distribution-0.7.zip and  unzip to mahout-distribution-0.7
1. rvm jruby-1.7.4
1. bundle install

# Demo

1. User based
  * ```ruby user_based.rb```
1. Item based
  * ```ruby item_based.rb```
1. Use MySQL
  * setup  https://github.com/tka/recommender-demo and import data 
  * ```ruby mysql.rb```
